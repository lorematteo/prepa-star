## Features

<img src="screenshots/flying-iphone.png" align="right" width="50%" hspace="10" vspace="10">

> Front End

*   [x] App Walkthrough
*   [x] Quiz Player
*   [x] Cosmetic Shop
*   [x] Math Formula LaTeX implementation
*   [ ] In-App Quiz Creation
*   [ ] Android version

*   [x] Account Creation
*   [x] Fetch new Quizzes in real time
*   [x] Players ranking
*   [x] Gamified XP and Money system
